# Xeno3ra | Portfolio

A macOS Startup Disk Selection-inspired portfolio site built with React, TypeScript, and Vite.

## Features

- **Boot screen** — logo + progress bar animation on load
- **Disk selector** — project icons slide in one by one, click to visit
- **Eject animation** — disks animate out before opening the link
- **About modal** — bio, stack, languages, and links
- **Keyboard navigation** — arrow keys / scroll wheel to browse, Enter to open
- **`?` shortcut** — keyboard shortcuts overlay
- **Circle contact bar** — GitHub, Stardance, and email with tooltips

## Stack

- React 18 + TypeScript
- Vite
- CSS Modules

## Dev

```bash
npm install
npm run dev
```

## Build
